# Privacy Note

### Overview
This note outlines how we handle data within the **SIAM** project. As this is currently a development-phase lab project for CS326, we prioritize data minimization and user transparency. 

### Data Collection & Usage
I have designed this application to collect only what is strictly necessary for its functionality:
* **Functional Data:** Any data entered into the application is used solely to demonstrate the project's features.
* **No Third-Party Sharing:** I do not sell, trade, or transmit any user data to external services or third-party trackers.

### Storage and Retention
* **Local Storage:** All data processed by this app is stored locally on the user's machine (e.g., in-memory or via local storage).
* **Retention Policy:** Since this is a demo, data is not intended for long-term storage. Users can clear their data at any time by refreshing the application or clearing their browser cache/local files.

### Security
I protect sensitive information (like API keys or environment variables) by using a `.env` file, which is explicitly excluded from my version control to prevent accidental exposure.

### User Rights
As the developer, I respect user privacy. Users have full control over the data they input and can delete it at any time. For the purposes of this lab, no permanent user profiles are created or maintained.
